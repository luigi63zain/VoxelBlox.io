# VoxelBlox.io
Play different blocky minigames such as Spleef and edit your character and username!

### Spleef
Eliminate the enemy bots by breaking the blocks under them! Don't fall in the lava!

### Freebuild
Double tap space to fly, build whatever you want to!

### Floor is Lava
Blocks randomly break, they regenerate after 20 seconds! Don't fall into the lava!
